# Dev Machine

Setup a development environment in a few seconds.

Requirements:

- [vagrant](https://www.vagrantup.com/)
- python [requests](http://docs.python-requests.org/en/latest/)

Usage:

`dev [OS]/[version] [packages...]`

Example:

`dev ubuntu/trusty64 haskell-stack`
